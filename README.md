# Kaiten Sushi Price Calculator 🍣

A lightweight, mobile-responsive web utility for real-time cost tracking at conveyor belt (Kaiten) sushi restaurants.

## 🎯 Project Goal
The goal is to eliminate the mental math involved in tracking your total spend while eating at sushi restaurants with tiered pricing (different colored plates). It provides a quick, intuitive interface to log plates as you eat, ensuring you stay within your budget.

## 🚀 Features
- **Restaurant Presets:** Quick-select pricing models for popular sushi chains (Sushiro, etc.).
- **Real-time Totals:** Instantly calculates total cost, including customizable tax/service charge.
- **Budget Tracking:** Visual indicators to show how close you are to your spending limit.
- **Mobile-First Design:** Optimized for one-handed use on a smartphone while at the restaurant.
- **Zero Dependencies:** Ultra-lightweight and fast loading.

## 🛠️ Tech Stack
- **Languages:** HTML5, CSS3, JavaScript (ES6+)
- **Architecture:** Single-file web application for maximum portability and speed.
- **Styling:** Custom Responsive CSS (No frameworks).
